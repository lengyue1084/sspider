# sspider
图片下载器

###########################################:
胖达图片采集器v0.1
该版本支持采集列表规则，页码{{n}}为变量
如：《https://www.58pic.com/collect/fav-{{n}}.html》
其中n为页码，起始页码一般默认为1
###########################################:
命令列表：
1、设置采集前缀 如：https://www.58pic.com/collect/fav-
2、设置采集后缀 如：.html,根据实际情况设置,默认为空
3、设置起始页码 如：1,默认值为1
4、设置最大页码 如：999,根据需要采集的页面列表采集设置默认为999
5、设置最大线程数默认8
6、开始采集
8、退出程序
